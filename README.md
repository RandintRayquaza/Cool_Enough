Cool Enough

Cool Enough is an imaginary luxury fashion brand concept built as a creative frontend experiment.

This project explores high-end fashion web design using smooth animations, cinematic scroll effects, and editorial-style layouts. It was created purely for fun and creative exploration.

About the Project

Cool Enough is designed as a luxury girlswear brand landing experience.
The goal was to experiment with:

Cinematic hero sections

Bento-style editorial layouts

Horizontal scroll effects

Premium hover interactions

Smooth animation flow

The design focuses on a clean, high-fashion aesthetic inspired by luxury campaign websites.

Tech Stack

This project was built using:

HTML

CSS

JavaScript

GSAP (GreenSock Animation Platform)

ScrollTrigger

Lenis (smooth scrolling)

Google Fonts

AI-generated images using Reve

All animations and layout interactions were implemented manually.

Features

Preloader animation

Animated hero section

Bento editorial grid

Horizontal scroll section with pinning

Scroll-based reveal animations

Hover overlays with smooth transitions

Cinematic image scaling effects

Image Generation

All model and fashion images were generated using Reve AI.

The visuals are created for experimental and conceptual purposes only.
